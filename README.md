# 智聘Pro招聘平台项目

## 🚀 项目简介

**智聘Pro**是一个基于大语言模型(LLM)和人工智能技术的新一代智能招聘服务平台，融合微服务架构和多级分销体系，致力于通过AI技术重塑招聘行业。

## ✨ 核心特色

### 🔥 技术亮点
- **LLM集成**: GPT-4/Claude等大模型驱动的智能对话和简历分析
- **AI算法**: 深度学习匹配算法，准确率提升40%+
- **微服务架构**: 高可用、高扩展的云原生架构
- **多级分销**: 创新的分销体系，激励用户推广和分享

### 🌟 产品优势
- **智能匹配**: 基于LLM的职位-简历语义匹配
- **多端支持**: 网页端、H5端、小程序端、App端全覆盖
- **实时推荐**: 毫秒级个性化职位推荐
- **智能分析**: 招聘成功率预测、人才流失预警

## 📱 多端应用

| 平台 | 特点 | 主要功能 |
|------|------|----------|
| **网页端** | 企业后台管理，数据分析报表 | 招聘管理、数据分析、分销管理 |
| **H5端** | 轻量级移动体验，快速访问 | 职位浏览、简历投递、分销推广 |
| **小程序端** | 微信生态集成，社交传播 | 微信分享、社交传播、分销激励 |
| **App端** | 原生性能体验，推送通知 | 推送通知、离线功能、分销工具 |

## 🏗️ 技术架构

### 前端技术栈
- **框架**: UniApp (支持多端发布)
- **UI组件**: uView UI
- **状态管理**: Pinia
- **构建工具**: Vite

### 后端技术栈
- **框架**: FastAPI (Python 3.9+)
- **数据库**: PostgreSQL + MongoDB + ClickHouse
- **缓存**: Redis Cluster
- **搜索引擎**: Elasticsearch
- **消息队列**: Apache Kafka + RabbitMQ
- **容器化**: Docker + Kubernetes

### AI技术栈
- **LLM模型**: GPT-4、Claude、文心一言、通义千问
- **算法框架**: TensorFlow、PyTorch、Scikit-learn
- **向量数据库**: Pinecone、Weaviate
- **模型部署**: ONNX、TensorRT

## 💰 商业模式

### 收入来源
- **企业服务** (60%): 会员费、职位推广、人才测评、招聘外包
- **求职服务** (25%): 高级简历、职业咨询、技能培训、面试辅导
- **分销佣金** (15%): 多级分销佣金分成、推广服务、培训服务

### 分销体系
```
一级分销商 (15%) → 二级分销商 (10%) → 三级分销商 (8%) → 四级分销商 (5%)
```

### 🚀 快速开始

#### 后端启动
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

#### 前端启动
```bash
cd frontend
npm install
npm run dev:h5
```

#### 访问地址
- **后端API**: http://localhost:8000
- **API文档**: http://localhost:8000/docs
- **前端H5**: http://localhost:3000
