# 旅游小程序项目

## 项目概述
基于大模型的智能旅游小程序，提供个性化旅游规划、智能问答、行程定制等服务。

## 文档
- [旅游小程序项目规划书](./旅游小程序项目规划书.md) - 完整的技术方案、功能清单、报价和进度计划

## 技术栈
- 后端：Python + FastAPI
- 数据库：PostgreSQL + Redis
- 大模型：OpenAI GPT-4/Claude 3
- 前端：微信小程序 + React管理后台
- 部署：Docker + Kubernetes
